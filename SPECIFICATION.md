# SPECIFICATION

Nama aplikasi: ScamEmotion  
Versi: 1.0  
Jenis: Prototaip aplikasi web amaran awal penipuan siber

## Fokus
ScamEmotion memfokuskan analisis pencetus emosi dan manipulasi emosi dalam mesej digital.

## Dataset
- `scamemotion_dataset.xlsx`

## Pencetus emosi
1. Ketakutan
2. Kecemasan
3. Harapan ganjaran
4. Kepercayaan palsu
5. Simpati
6. Rasa bersalah

## Paparan utama
1. Semak Mesej Mencurigakan
2. Keputusan Keseluruhan
3. Analisis Pencetus Emosi
4. Frasa dan Petanda Dikesan
5. Padanan Data Kawalan Sepadan
6. Cadangan Tindakan Selamat
7. Penafian
