# ScamEmotion 1.0

ScamEmotion ialah aplikasi khusus untuk menganalisis pencetus emosi dan manipulasi emosi dalam mesej digital.

## Fail utama
- `app.py`
- `requirements.txt`
- `scamemotion_dataset.xlsx`
- `README.md`
- `SPECIFICATION.md`

## Cara jalankan
```bash
pip install -r requirements.txt
streamlit run app.py
```
