# ScamEmotion 1.1

## Perubahan 1.1
- Julat tahap risiko ScamEmotion dilaraskan:
  - 0–24 = Rendah
  - 25–59 = Sederhana
  - 60–79 = Tinggi
  - 80–100 = Sangat Tinggi
- Logik kritikal ditambah:
  - Ketakutan + Kecemasan + arahan berisiko = skor minimum 85.
- Bahagian lain dikekalkan.

## Cara jalankan
```bash
pip install -r requirements.txt
streamlit run app.py
```
